# Morse Code Generator and Translator

A **Morse Code Generator and Translator** application that allows users to **encode** text into Morse code and **decode** Morse code into text. The frontend is built using **ReactJS**, and the backend is developed with **Spring Boot**.

## Features

- **Text-to-Morse Conversion (Encoding)**: Input plain text and get the corresponding Morse code output.
- **Morse-to-Text Conversion (Decoding)**: Input Morse code and get the corresponding plain text output.
- **Responsive UI**: Built with ReactJS to ensure a smooth and responsive experience across devices.
- **Spring Boot Backend**: Handles text-to-Morse and Morse-to-text conversions and serves the application via API endpoints.
- **Easy to Use**: Simply select the desired action (encode or decode), type in the input, and the system will automatically generate the result.

## Tech Stack

- **Frontend**: ReactJS (with hooks, functional components, etc.)
- **Backend**: Spring Boot (Java-based REST API)
- **Database**: (Optional) If you plan to persist data like user input, can integrate a database (e.g., MySQL).
- **Morse Code Logic**: Implemented on the backend to translate characters to Morse code and vice versa.

## Prerequisites

To run this project locally, you will need the following installed:

- **Node.js** and **npm** (for running the frontend)
- **JDK 11 or later** (for running the Spring Boot application)
- **Maven** (for building and running the Spring Boot app)
